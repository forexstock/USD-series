## What is USD Series?

USD Series is a suite of eight purpose-built stablecoins, all issued from a single trusted account on the Stellar network. Each token shares the same dollar parity and issuer guarantee, while serving a distinct role вҖ” from payments and anchoring to minting, industry, and beyond.

Think of it as a modular dollar infrastructure for Stellar.

```
Issuer  вҶ’  GC5QAUERATRRIPZWOWBEDTIRH6E3ROP2H7ZJWZYJNXPDMM3R27WLUSDC
Domain  вҶ’  https://usdseries.vercel.app/
Network вҶ’  Stellar Public Network
Peg     вҶ’  1 token = 1 USD
```

## Add a Trustline

To hold any USD Series token, add a trustline from your Stellar wallet. Replace `USDPAY` with whichever token you'd like to receive.

```
Asset code   : USDPAY  (or USDANCHOR, USDCORE, USDCROWN, USDINDUS, USDMINT, USDSTELLAR, USDXLM)
Asset issuer : GC5QAUERATRRIPZWOWBEDTIRH6E3ROP2H7ZJWZYJNXPDMM3R27WLUSDC
```

Most Stellar wallets (Lobstr, Solar, Freighter) let you add a trustline by searching the asset code and pasting the issuer address.
